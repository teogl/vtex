## Installation BACK

```bash
$ yarn install
```

or 

```bash
$ npm install
```

## Running the app

- copy .env.example variable environment to .env file

```bash
# development
$ docker-compose up -d

```

```bash
# development
$ npm run dev or yarn dev

```

- open url: http://localhost:3000/documentation
